## GUÍA DE GENERACIÓN

### Herramientas Recomendadas

| Herramienta | Mejor Para | Precio |
|-------------|------------|--------|
| **Midjourney v6.1** | Fotografía, retratos, composiciones | $10-60/mes |
| **DALL-E 3** | UI mockups, interfaces, texto en español | $20/mes (ChatGPT+) |
| **Leonardo AI** | Producto, mockups 3D | Free tier / $12+/mes |
| **Ideogram 2.0** | Texto legible, logos, branding | Free tier / $7+/mes |
| **Flux Pro** | Realismo fotográfico extremo | Via API |

### Tips para Mejores Resultados

1. **Para UI/Interfaces**: Usar DALL-E 3 o Leonardo - mejor manejo de texto español
2. **Para Retratos**: Midjourney v6.1 con `--style raw` para fotorealismo
3. **Para Productos/Mockups**: Leonardo AI con modo "Photography"
4. **Para 3D/Tech**: Midjourney con `--v 6.1` y términos como "octane render"

### Parámetros Midjourney Recomendados

```
--ar [ratio]     # Aspect ratio (4:3, 16:10, 1:1, 4:5)
--v 6.1          # Versión más reciente
--style raw      # Fotografía más realista
--q 2            # Calidad máxima
--s 250          # Stylization medio-bajo para realismo
```

### Post-Procesamiento

1. Upscale a resolución final con Topaz Gigapixel o similar
2. Convertir a WebP con calidad 85%
3. Crear versiones @2x para retina
4. Optimizar con TinyPNG/Squoosh para web
